# Superstore Sales Analysis

## Project Overview

This project analyzes Superstore sales data to uncover business insights related to sales performance, profitability, customer behavior, and regional trends. The analysis was performed using Python, SQL, and Power BI to support data-driven decision-making.

## Problem Statement

Businesses generate large amounts of sales data, but without proper analysis it is difficult to identify growth opportunities, profitable products, and areas requiring improvement. This project aims to transform raw sales data into actionable business insights.

## Objectives

- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Discover high-revenue and high-profit regions
- Evaluate the impact of discounts on profitability
- Understand customer purchasing patterns
- Build an interactive dashboard for business reporting

## Tools & Technologies

  - Pandas
  - NumPy

## Data Analysis Process

### 1. Data Cleaning
- Handled missing values
- Checked for duplicate records
- Verified data types and consistency

### 2. Exploratory Data Analysis (EDA)
- Sales trend analysis
- Profit analysis
- Category and sub-category performance
- Regional performance comparison

### 3. SQL Analysis
- Revenue calculations
- Profitability analysis
- Top-performing products
- Regional sales insights

### 4. Dashboard Development
Created an interactive Power BI dashboard featuring:
- Total Sales
- Total Profit
- Sales by Category
- Sales by Region
- Monthly Sales Trends
- Top Products Analysis

## Key Insights

- Technology category generated the highest profit.
- Certain products generated high sales but low profit margins.
- Some regions consistently outperformed others in both sales and profitability.
- Excessive discounting negatively impacted profit in multiple categories.
